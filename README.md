### Configuration

Make file in root directory, `config.json`

```
{
  "prefix": "",
  "token": "",
  "ownerid": "",
  "defaultGame": "for DMs",
  "debug": false,
  "guildID": "",
  "role": {
    "admin": ""
  },
  "channel": {
    "log": "", <----- This must be channel name
    "modmail": "" <--- This must be channel ID
  }
}
```
